## Hi there 👋

I wanted to make a GitHub page to have a place to learn and teach aspects of coding that are important to me.
Nothing written on this page uses AI.
It's certainly not perfect.
It's probably not even good.
But then again, that's the point.
The only way to learn is to do, and doing something requires you to be bad for quite some time.
So enjoy the process of being bad.
Just as you can't force yourself to be good without practice,
you can't be bad forever if you try.
